# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.9] - 2025-02-05

Upgraded ag grid library to latest version, reducing bundle size and improving performance.

## [0.0.8] - 2024-09-27

Upgraded ag grid library to latest version. This ugrade included access to better keyboard naviagition and
updates to the row selection api.

## [0.0.7] - 2024-08-05

Updated column defs to include a tooltip method for setting a custom text in a column tooltip

## [0.0.6] - 2024-07-23

Updated column defs to include a value formatter so that we can modify table cells if we wish

## [0.0.5] - 2024-06-25

Added ability to show tooltips per column and made them off by default

## [0.0.4] - 2024-06-21

Extended the props to allow the Box props for the wrapper of the SSData Table component. This
will prevent us from having to create other parent components because we couldn't pass styles.

## [0.0.3] - 2024-06-17

Added rowClassRules support, theming for disabled rows, tweaked selected row colour and made grid spacing smaller

## [0.0.2] - 2024-06-06

Added column size option and fixed resizing behaviour

## [0.0.1] - 2024-05-28

Migrate to library from 'discovery/snmp-credentials'
