# SelectTable

A table that can be used in a view-only mode, single or multi-selection.

## How to use

1. Import the table `import {SelectTable} from "@statseeker/components/SelectTable`
2. Start using it!

### Stylesheet

A LESS stylesheet is exposed by the package.

1. import the stylesheet: `@import "style" from "@statseeker/components/SelectTable`;

## Unit tests

1. run the unit tests with `nx run shared-select-table:test`
